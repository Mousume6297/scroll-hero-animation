# Scroll Hero Animation

A scroll-driven hero section animation built using HTML, CSS, JavaScript and GSAP.

##  Features
- Smooth intro animation using GSAP
- Scroll-based image movement
- Hero section with headline and statistics
- Smooth and performant animations

##  Tech Stack
- HTML
- CSS
- JavaScript
- GSAP
